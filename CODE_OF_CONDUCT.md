# Code of Conduct

Servy follows the Contributor Covenant Code of Conduct, version 2.1.

## Our pledge

We pledge to make participation in this project respectful and harassment-free for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but prioritize user safety and correctness.
- Accept maintainer decisions about security and scope.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Publishing private vulnerability details before maintainers have had a chance to respond.
- Pressuring maintainers to merge unsafe changes.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or ban contributors who violate this code.
